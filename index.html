¿ Qué es diseño responsivo ?

Se define como Diseño Responsivo Web o por sus siglas en ingles RWD (Responsive Web Design).

Una forma de desarrollar sitios web asegurando, que el diseño está optimizado para que el formato se ajuste en relación al tamaño de la pantalla del dispositivo mediante el uso de las siguientes técnicas:

  • Un grid flexible en esta ocasión hablaremos de 960gs.
  • Imágenes flexibles que se ajustan a un contexto fluido.
  • CSS3 Media queries, que optimizan el diseño para diferentes contextos.

Diferentes maneras de proveer soluciones móviles.

  • Aplicaciones Nativas.
  • Aplicaciones Web HTML5 & CSS3.
  • Versión móvil del sitio.
  • Diseño responsivo (RWD mobile first).
  • Diseño responsivo + componentes del lado del servidor (RESS).

Diseño Responsivo vs Móvil
(Soluciones del lado del servidor)

En el diseño responsivo los URL del sitio son consistentes a través de diferentes dispositivos y el sitio adapta su contenido en función de las capacidades del navegador en el que lo muestra.

En las soluciones del lado del servidor, por el contrario, sólo se envía lo que el cliente necesita optimizado por dispositivo. Los recursos (imágenes, javascript, css styles) pueden ser optimizados antes de llegar al navegador. Es decir se sirve un diferente grupo de plantillas y recursos ya que servidor direcciona el sitio en base al dispositivo que consulta el sitio.

Diseño responsivo + componentes del lado del servidor (RESS)

RESS combina el diseño adaptivo con componentes del lado del servidor optimizando evitar servir todo el sito.

Es decir es posible mantener una estructura genérica y compartida entre diferentes dispositivos, pero los componentes o bloques dentro del sitio cuentan con implementaciones en el lado del servidor en base a los tipos de dispositivos que generan solo el markup requerido.

¿ Cuando es necesario pensar en diseño responsivo ?

  • Estas pensando rediseñar o iniciar un sitio desde cero. Es necesario estalecer esto al momento de elegir el tema base sobre el cual se implementará el sitio.
  • Quieres reducir costos de implementación y mantenimiento. Es mas caro mantener una versión móvil del sitio ya que practicamente existe la necesidad de mantener dos sitios por separado. En el caso de un sitio responsivo se comprate código.
  • Quieres soportar nuevos dispositivos cuando estos aparezcan en el mercado.

¿ Qué es y como funciona un sistema de grid como 960 ?

Un CSS Grid Framework es una herramienta que nos facilita el maquetado y el desarrollo de un sitio web al proveernos una guía sobre la cual trabajar.

Esta guía consiste en un sistema de columnas, espacios y márgenes que nos ayudan a posicionar los bloques de contenido de nuestro diseño de manera rápida y organizada.

¿CSS3 Media queries que son y como funcionan ?

Son instrucciones condicionantes para determinar que hoja de estilo ó cuales clases dentro de esta seran las que se utilizen. 

<link rel="stylesheet" href="style.css" 
 media="(min-width: 960px)">
@media screen and (min-width: 960px) { 
// definición de clases y atributos.
}

¿Cuales son los Viewports mas comunes basados en los dispositivos actuales ?

Unidad de medida en pixeles

  • 320x480 - Smartphone portrait.
  • 480x320 - Smartphone landscape.
  • 540x960 - Smartphone largerscreen portrait.
  • 960x540 - Smartphone largerscreen landscape.
  • 768x1024 - Tablet portrait
  • 1024x768 - Tablet landscape, Netbook, Laptop & Desktop.
  • Mayor 1024x768 - Laptop & Desktop.

Breakpoints

  • 0-519 pixels: La mayoría de los telefonos en formato landspace, se podría ajustar el formato portrait con un ancho fluido CSS.
  • 520-759 pixels: Teléfonos con pantallas amplias y resoluciones altas en formato landspace ó Tablets con resoluciones bajas o pantallas pequeñas en formato portrait.
  • 760-959 pixels: La mayoria de Tablets en formato landspace.
  • 960-1279 pixels: Tablets con pantallas alta definición, computadoras portatiles "laptops" y dispositivos de escritorio "desktop".
  • 1280+: Monitores grandes o con resoluciones altas y televisiones inteligentes "Smart TV".

    ¿Que puede hacer Omega por mí ?

    • Estas pensando rediseñar o iniciar un sitio en Drupal que se ajuste y pueda servir contenido a diferentes dispositivos a la vez, utilizando el mismo código base ó simplemente deseas tener las bases para poder generar un sitio responsivo en Drupal.

    ¿ Qué ofrece Omega Theme ?

    Hay muchas ventajas estéticas y de diseño que obtendremos al usar un framework:

    • Uniformidad y consistencia en la posición de los elementos.
    • Elimina el uso de tablas.
    • Soporte para múltiples navegadores y pantallas.
    • Provee una base de css con la cual trabajar rápidamente, ahorrando el tiempo necesario para construir una.
    • Cohesion visual entre los elementos de la página.
    • Fácil de implementar y mantener.

    ¿ Cómo ocultar un bloque cuando la medida es menor a la de una "tablet" ?

    Omega utiliza un método llamado "Mobile first" para responder ante las diferentes medidas de los dispositivos. Para ocultar el bloque por defecto utilizamos la hoja de estilo llamada global.css

    .flexslider { display:none; }

    Para mostrarlo en resolución igual ó mayor a una "tablet" utilizamos la hoja llamada theme_name-alpha-default-narrow.css

    .flexslider { display:block; }
